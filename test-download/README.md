## Optik Ecosystem

Optik Ecosystem is a full-stack, on-chain infrastructure platform designed to enable non-technical and technical users alike to launch production-ready blockchain projects with real economic depth.

### Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Environment Variables

Copy `.env.local.example` to `.env.local` and fill in your Solana network and RPC host.
